# スパイスルニンジャゲーム

6さいの子供が考えたゲームを、そのまま形にしたスマホ向けブラウザゲーム。

▶ あそぶ: https://yurukusa.github.io/supaisuru-ninja/

## あそびかた
1. そらから地獄みたいな古いお屋敷がドーンと降ってくる。忍者は中に入る
2. 指で忍者を動かして、敵にぶつかると算数クイズ（4+4はな〜んだ？）
3. 正解すると敵を倒して、にんじゃカード（技）がもらえる
4. 5回連続で正解するとボスが出てくる
5. ボス戦はハンマー。画面のマークをタップして15回叩けば勝ち。ボスも急に叩いてくるから、15回叩かれたら負け
6. にんじゃカードの技を使えば10回で倒せる。ただし時間切れになると技は消える（与えたダメージは残る）
7. ボスを倒すとキラカード（5回で倒せる技＋エモート付き）とスタンプがもらえる。スタンプは指でドラッグして押す
8. カードは全部で100種類。図鑑でコレクションできる
9. キラカードを5枚集めると、お屋敷の地下の封印が解けて…スクリーンクスモンスターが目を覚ます

## 技術メモ
- 素材は過去作 ninja-math / ninja-hammer のスプライト・効果音・BGMを流用
- BGMはm4a（iOS Safari対応）。効果音はWebAudio
- セーブはlocalStorage。スクリーンクスモンスターと屋敷背景はCanvas手続き描画
- フレームワークなしの素のHTML/CSS/JS
