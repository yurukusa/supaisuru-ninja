# スパイスルニンジャゲームの制作計画

原案は6歳の子供。仕様は会話に全部ある。忠実に作る。

## 仕様の要点
- タイトルは「スパイスルニンジャゲーム」
- イントロ：忍者がいる→でかいお屋敷（地獄風・古い時代）がドーンと降ってくる→中に入る
- 屋敷内：指で忍者を動かす。敵に触れると算数クイズ（4+4など）
- 正解で敵を倒す→ノーマル忍者カード（技として使える）
- 5回連続正解でボス出現
- ボス戦：ハンマー15回で勝ち。画面のマークをタップで叩く。ボスも急に叩いてくる。15回叩かれたら負け
- ノーマルカード技＝10回で倒せる。ただし時間切れあり（切れたら技消滅、ダメージは残る）
- キラカード技＝5回で倒せる。時間切れなし。ボス撃破ごとに入手。エモート付きで忍者が実演してくれる
- 技は画面上部に表示、タップで使う
- 勝ったら丸数字が青くなり、スタンプをドラッグして押す（スタンプラリー）
- カードは全部で約100種（ノーマル95＋キラ5）。図鑑あり
- キラ5枚コンプで地下の封印が解けてスクリーンクスモンスター登場（影に包まれた黒い体、紫緑青赤の目がたくさん）
- スクリーンクス戦はキラの技のタップのみで攻撃
- 倒したらエンディング。最初からやり直せる。セーブはlocalStorage
- GitHub Pagesで即プレイ（スマホ縦持ち）
